# Деплой на Vervel
[Vercel](https://todo-psi-seven.vercel.app/).


## Команды
Для установки пакетов используйте команду npm i
### Запуск сервера для разработки

```shell
npm run start
```

### Сборка проекта без оптимизации

```shell
npm run build-dev
```

### Сборка проекта с оптимизацией

```shell
npm run build-prod
```

### Очистка папки dist

```shell
npm run clear
```
